**TestingWebApp** — это проект, разработанный на Django, который имитирует форму регистрации для демонстрации инструментов автоматизированной валидации и поиска уязвимостей в текстовых данных. 

## 🚀 Основные фичи

Проект реализует комплексную проверку по следующим критериям:

* Поиск паттернов SQL-инъекций (`SELECT`, `DROP`, `--`) и XSS-уязвимостей (HTML-теги, скрипты).
* Проверка граничных значений (минимальная и максимальная длина строки).
* Обнаружение смешивания алфавитов (кириллица + латиница) и скрытых спецсимволов.
* Запрет на использование цифр и недопустимых знаков в полях имен.
* Интуитивно понятный интерфейс с цветовой индикацией результатов.
